# Aplicación de Inscripción de Estudiantes
***
### Módulos

* express
* mysql2 
* sequelize
* cors
* dotenv
* jsonwebtoken
* bcryptjs

## Uso del respositorio

```bash
npm i express mysql2 sequelize cors dotenv jsonwebtoken bcryptjs

```


## Inicio del proyecto

### Opción 1
```bash
npm run dev

```
### Opción 2
```bash
git clone https://github.com/joseleonardo1995/Node_Grupo2_Estudiantes.git
npm i
npm run dev
```
